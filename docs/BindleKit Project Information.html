    Bindle Binaries Objective-C Kit

    The Bindle Binaries Objective-C Kit is a collection of Objective-C classes that were created to fulfill a need in a particular project but may be of use in additional iOS or OS X projects. Example of resuable classes contained within the Bindle Binaries Objective-C Kit include BKSplitViewController, BKReachability, and BKPosixRegex.
