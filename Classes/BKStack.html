Overview

Implements a generic last in, first out (LIFO) data structure.

BKStack is a class which stores a series of objects in a stack data structure. This class supports standard stack operations. The class is designed to be re-usable and does not depend upon project specific classes.

Tasks

Creating and Initializing a Stack

  • – init

    Initializes an instance of BKStack.

Adding Objects

  • – dup

    Pops the top object of the stack and pushes it onto the stack twice. This method retrieves the last object added to the stack and pushes the object onto the stack. This creates two instances of the object on the stack.
  • – push:

    Inserts an object to the top of the stack.

Removing Objects

  • – pop

    Removes the last object added to the stack. The object added to the stack immediatly prior to the removed object becomes the new top of the stack.

Querying a Stack

  • – count

    Returns the number of objects currently in the stack.
  • – empty

    Returns a BOOL which indicating if the stack currently contains objects.
  • – top

    Returns the last object added to the stack. Returns the last object added to the stack without removing the object from the stack.

Rearranging Content

  • – leftRotate

    Moves the object at the bottom of the stack to the top of the stack. To fill the gap, all other objects are moved one position closer to the bottom of the stack.
  • – rightRotate

    Moves the object at the top of the stack to the bottom of the stack. To create a gap for the object, all other objects are moved one position closer to the top of the stack.
  • – swap

    Switches the position of the top two objects in the stack.
